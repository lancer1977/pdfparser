# Core Capabilities

## Summary
pdfparser provides foundational capabilities that extend beyond its primary user interface.

## Value Provided
- Enables automation within the pdfparser domain.
- Provides shared logic and utilities.


## Newly Documented Existing Capabilities
- **pages**: render full pages to PNG (recommended for “what it looks like”)
- **images**: extract embedded image objects, then **organize + filter**
- **both**: do both

### Beyond the App
- Docker Containerization
